Why Websockets are Cool

by Yuqing Wu and Annabel Zhang


What are Websockets?

  • Websockets establish a connection between the backend server and the user, allowing the website to asynchronously stream information to and from the server in realtime
  • There is a websockets library in Python and JavaScript to help create websocket connections
  • Websockets can listen for events on the website to run functions (click on element or always active)
  • They send and receive information using JSON files

A Brief History

  • Created alongside The Websockets API made by W3C (an internet organization led by Tim Berners-Lee, the “inventory of the World Wide Web”)
  • When the web was new, people used HTTP long polling
    • HTTP long polling - a half duplex system of call and response
    • A new request/response call had to be made every time the client wants to communicate with the server
